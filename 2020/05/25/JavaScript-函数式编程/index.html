总结

​ 一种用于构建 JavaScript 应用程序的技术称为 “函数式编程”.

  • 确定抽象, 并为其构建函数
  • 利用已有的函数来构建更为复杂的抽象
  • 通过将现有的函数传给其他的函数来构建更加复杂的抽象
