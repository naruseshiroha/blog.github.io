～よう(用)

みず　＆　おゆ

水をください。
お湯をください。

动词ます「连用」形 + もの → …的东西

食べ物「食物」 飲み物「饮料」 買い物「购物」 忘れ物「遗失物」

海

  • うみ「训读」，海辺(うみべ)０[海边，海滨]
  • かい「音读」，海外(かいがい)１[海外，国外]

山

  • やま「训读」，山田(やまだ)０[山田]
  • さん「音读」，富士山(ふじさん)１[富士山]
  • ざん「音读」，火山(かざん)１[火山]

紙

  • かみ「训读」，紙飛行機(かみひこうき)４[纸飞机；M]
  • し「音读」，新聞紙(しんぶんし)３[报纸]

気持ち

気持ちがいい 気持ちが悪い きもい[口语]

暑い|熱い|厚い

  • 暑い　→　天气热　→　暑い天気
  • 熱い　→　温度高　→　熱いスープ
  • 厚い　→　厚的　→　厚い服

寒い|冷たい|涼しい

  • 寒い　→　天气冷　→　寒い冬
  • 冷たい　→　冰冷的　→　冷たい水
  • 涼しい　→　凉爽的　→　涼しい風

易しい|優しい

易しい試験　優しい先生

あら|あれ

在感到惊讶或疑惑不解时使用的词语

  • あら：女性专用
  • あれ：男女通用

～用

名词(使用者・用途) + 用　→　供...使用

  • 使用者
    • 子供用　大人用　女性用　お客様用　
  • 用途
    • 試験用　ワイン用のグラス
