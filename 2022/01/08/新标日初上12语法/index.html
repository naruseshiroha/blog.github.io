新标日初上12语法

投稿日 2022-01-08 編集日 2022-04-05 カテゴリ 新标日 単語数： 2k 読書の時間 ≈ 3 分

12 李さんは　森さんより　若いです

名1は　名2より　形1/形2です

李さんは　森さんより　若いです。

より：格助词，表示比较的基准。
...は…より…：译为”…比…(更)…”

example

10月/9月/涼しい　→　
10月は9月より涼しいです。

練習

森さん/吉田さん/元気　→　
森さんは吉田さんより元気です。

電車/バス/速い　→　
電車はバスより速いです。

补充

ずっと　→　副词，表示两者相差悬殊，译为”…得多”

中国は日本よりずっと広いです。
李さんは小野さんよりずっときれいです。

名2より　名1の　ほうが　形1/形2です

李さんは　森さんより　若いです。
森さんより　李さんの　ほうが　若いです。

が：主格助词，强调主语
ほう：名词，表示”一边、一方”
…より…のほうが…：译为”比起…(来说)，要(更)…些”

example

10月は9月より涼しい。
9月より10月のほうが涼しい。

奈良は京都より静かです。
京都より奈良のほうが静かです。

名1は　名2ほど　形1/形2否定形

李さんは　森さんより　若いです。
李さんは　森さんほど　若くないです。

ほど：副助词，表示程度
…は…ほど…：译为”…不如…”，”没(那么)…”

example

神戸は大阪ほどにぎやかではありません。
今日は昨日ほど暑くないです。

練習

バス/地下鉄/便利　→　
バスは地下鉄ほど便利ではありません。

名1と　名2と　どちらが　形1/形2ですか

李さんと　森さんと　どちらが　若いですか。

どちら：两者间比较的疑问句使用该疑问词
…と…と　どちらが　…：译为”…和…，哪个更…”

口语形式：「どっち」
李さんと森さんとどっちが若いですか。
「森さんより」李さんのほうが若いです。

example


リンゴとミカンとどちらが高いですか。
リンゴのほうが高いです。

練習


森さんと吉田さんとどちらが歌が上手ですか。
森さんのほうが上手です。


飛行機と新幹線と、どちらが速いですか。
飛行機のほうが速いです。


李さんと小野さんとどちらがきれいですか。
どちらもきれいです。

どちら＋も＋肯定[全面肯定句]

練習


テニスとサッカーとどちらが好きですか。
どちらも好きです。


英語と日本語とどちらが難しいですか。
どちらも難しいです。

な１(の中)で　名2がいちばん　形1/形2です

回顾：「で」的用法

  • 1.交通工具
    飛行機で日本へ行きます。
  • 2.表示动作发生的场所
    図書館で勉強します。
  • 3.表示手段、工具、原材料
    ペンで名前を書きます。
  • 4.表示限定范围
    日本料理の中で

注意：
1).非时空的名词 + の中で
スポーツの中で 音楽の中で
2).时空/空间的名词 + で
1年で 日本で

日本料理の中で寿司がいちばんおいしいです。
1年で2月がいちばん寒いです。

…(の中)で…がいちばん…：译为”在…之中，…最…”

example

クラスの中で李さんがいちばんきれいです。
日本で富士山がいちばん高いです。

补充

～の中でいちばん～[形]名1は名2です。
译为：”在…之中，最…的(人/物)是…”

example

日本料理の中で寿司がいちばんおいしいです。
日本料理の中でいちばんおいしい料理は寿司です。

練習

クラスの中で李さんがいちばんきれいです。　→
クラスの中でいちばんきれいな人は李さんです。

日本で富士山がいちばん高いです。　→
日本でいちばん高い山は富士山です。

～(の中)で　疑问词が　いちばん　形1/形2ですか

季節の中で　いつが　いちばん　好きですか。
～(の中)で　～[疑问词]が　いちばん　[形容词]ですか：译为：”在…之中，…是最…的呢?”

  • 疑问词
    • どの名/何/どれ　→　物
    • だれ　→　人
    • いつ　→　时间

example

飲み物の中で何がいちばん好きですか。
ウーロン茶がいちばん好きです。

会社の中でだれがいちばん忙しいですか。
社長がいちばん忙しいです。

季節の中でいつがいちばん暑いですか。
夏がいちばん暑いです。

何：用于从一个大概范围中选出一个的情况
食べ物の中で何がいちばん好きですか。
リンゴがいちばん好きです。

どの＋名　＆　どれ：用于被选择的十五存在于眼前的情况或已见于上下文
この中でどの料理がいちばん好きですか。
この料理がいちばん好きです。

リンゴとナシとバナナの中でどれがいちばん好きですか。
リンゴがいちばん好きです。

練習


季節の中でいつがいちばん好きですか。
夏がいちばん好きです。


果物の中で何がいちばんおいしいですか。
イチゴがいちばんおいしいです。


スポーツの中で何がいちばん上手ですか。
サッカーがいちばん上手です。

补充

クラスの中でだれがいちばんきれいですか。
李さんがいちばんきれいです。
↓
クラスの中でいちばんきれいな人はだれですか。
李さんです。

練習

A：今天去荞麦面馆吗?
A：今日は蕎麦屋へ行きますか。
B：嗯，好啊。
B：ええ、いいですね。
A：在这些店里，哪家店最便宜呢?
A：この店の中で、どの店がいちばん安いですか。
B：这家店最便宜。而且很好吃哦。
B：この店はいちばん安いです。そして、とてもおいしいですよ。
A：你要吃什么呢?
A：何を食べますか。
B：吃寿司和荞麦面。
B：寿司とそばを食べます。
A：寿司和荞麦面那个更好吃呢?
A：寿司とそばとどちらがおいしいですか。
B：都挺好吃的哟。
B：どちらもおいしいですよ。
