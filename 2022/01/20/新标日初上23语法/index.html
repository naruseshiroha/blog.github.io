新标日初上23语法

投稿日 2022-01-20 編集日 2022-03-23 カテゴリ 新标日 単語数： 2.2k 読書の時間 ≈ 4 分

23 休みの　日、散歩したり　買い物に　行ったり　します

～たり～たり

  • 接续
    • 动词：た形 + たり
      行く → 行ったり 起きる → 起きたり
    • 一类形容词：词尾い + かったり
      寒い → 寒かったり 高い → 高かったり いい → よかったり
    • 二类形容词：词干 + だったり
      静か → 静かだったり 簡単 → 簡単だったり
    • 名词：词干 + だったり
      日本人 → 日本人だったり 晴れ → 晴れだったり

动词たり　动词たりします

  • 1）表示从众多的动作中部分列举出两个或多个同类有代表性的动作。
    译为”有时…有时”、”又…又…”、”…啦…啦”。
    休みの日、散歩したり買い物に行ったりします。
    • example
      ここでお酒を飲んだり、タバコを吸ったりしないでください。
      昨日公園で写真を撮ったり散歩したりして、とても楽しかったです。
      母は毎日家で料理を作ったり、洗濯したり、掃除したりします。
    • 注意
      前接动词必须是性质相同的同一类的动作
      昨日、散歩に行ったり、買い物に行ったりしました。（〇）
      昨日、顔を洗ったり、買い物に行ったりしました。（×）
    • 練習 毎晩何をしますか。
      本を読んだりテレビを見たりします。 日曜日、何をしましたか。
      部屋を掃除したり車を洗ったりしました。 仕事が終ってから、何をしすか。
      飲みに行ったりカラオケに行ったりします。
  • 2）表示性质相反的动作反复交替发生。多是一对反义词的对举，或者是同一动词的肯定否定形式。
    译为”一会儿…一会儿…”、”有时…有时…”。
    • example
      テレビをつけたり消したりしないでください。
      雨が降ったり、止んだりしています。
      家の前をたくさんの車が行ったり来たりしています。
      本を読んだり読まなかったりしています。

形1/形2たり　形1/形2たりです

  • 表示列举多种状态或事物状况交替反复交换不定。多为一组反义词。或是同一词语的肯否定形式。
    译为”一会儿…一会儿…”、”有时…有时…”。
    チケットは高かったり安かったりです。

  • example
    今日の気持ちはよかったり悪かったりです。
    天気はよかったりよくなかったりです。
    公園は静かだったりにぎやかだったりです。
    部屋はきれいだったり汚かったりです。
    部屋はきれいだったりきれいじゃなかったりです。

  • 練習
    試験/易しい/難しい　→
    試験は易しかったり難しかったりです。

    閉店時刻/早い/遅い　→
    閉店時刻は早かったり遅かったりです。

    仕事/暇/忙しい　→
    仕事は暇だったり忙しかったりです。

名词たり　名词たりです

  • 从许多的事物中举出若干事例。
    译为”有时…有时…”、”或…或…”。
    日本語の先生は中国人だったり日本人だったりです。
  • example
    帰りは電車だったり、バスだったりです。
    休みは土曜日だったり、月曜日だったりです。

小句 + か

简体小句 + か →表示不确定的内容

  • 接续
    • 动词/形1 简体
    • 形2/名词 简体(「だ」省略)

小句(不含疑问词) + かどうか

森さんは中国へ行きますか、行きませんか。
私は分かりません。

大主语「は」小主语「が」
私は森さんが中国へ行くかどうか分かりません。

昨日李さんは休みました。　→
昨日李さんが休んだかどうかわかりません。

この本はおもしろいです。　→
この本がおもしろいかどうか教えてください。

新しい先生はきれいです。　→
新しい先生がきれいかどうか知りたいです

昨日銀行は休みでした。　→
昨日銀行が休みだったかどうかわかりません。

补充说明

李さんが日本へ行くかどうかわかりません。
どう：相当于前接词语相应的否定形式。「いかない」

example

昨日李さんが休んだかどうかわかりません。　→
昨日李さんが休んだか休まなかったかわかりません。

この本がおもしろいかどうか教えてください。　→
この本がおもしろかおもしろくないか教えてください。

新しい先生がきれいかどうか知りたいです。　→
新しい先生がきれいかきれいじゃないか知りたいです。

昨日デパートは休みだったかどうか分かりません。　→
昨日デパートは休みだったか休みではなかったか分かりません。

example & 練習

今日は森さんは来ます。　→
今日は森さんが来るかどうか分かりません。
今日は森さんが来るか来ないか分かりません。

明日雪が降ります。　→
明日雪が降りるかどうか分かりません。
明日雪が降りるか降らないか分かりません。

来週暇です。　→
来週暇かどうか分かりません。
来週暇か暇じゃないか分かりません。

小句(含疑问词) + か

森さんはいつ中国へ行きますか。
私は分かりません。

大主语「は」小主语「が」
私は森さんがいつ中国へ行くか分かりません。

昨日何をしましたか。　→
昨日何をしたか忘れました。

会社でだれがいちばん忙しいですか。　→
会社でだれがいちばん忙しいか分かりますか。

どうして夏が好きですか。　→
どうして夏が好きか教えてください。

これはだれの本ですか。　→
これがだれの本か分かりません。。

example & 練習

森さんは何時ごろ来ますか。　→
森さんが何時ごろ来るか知っていますか。

卒業式は何時に始まりますか。　→
卒業式が何時に始まるか知っていますか。

だれが会社の車を運転していますか。　→
だれが会社の車を運転しているか知っていますか。

練習

A：知道明天休不休息吗?
A：明日が休みかどうか知っていますか。
B：因为明天星期日，所以休息。
B：明日は日曜日ですから、休みですよ。
A：那请告诉我要做什么?
A：じゃあ、何をするか教えてください。
B：在家扫除啦，去超市买东西啦什么的。
B：家で掃除をしたり、スーパーへ買い物に行ったりします。
