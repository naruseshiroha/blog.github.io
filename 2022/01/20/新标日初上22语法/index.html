新标日初上22语法

投稿日 2022-01-20 編集日 2022-04-05 カテゴリ 新标日 単語数： 1.4k 読書の時間 ≈ 2 分

敬体和简体的区别

敬体 简体
①何时何地、对何人，均可以使用敬体。 ①亲密的朋友、同事、家人间使用简体。
②书信使用敬体 ②报纸、书籍、论文、日记等使用简体。

敬体和简体

一类形容词谓语形式简体

敬体 简体
忙しいです 忙しい
忙しくないです 忙しくない
忙しかったです 忙しかった
忙しくなかったです 忙しくなかった

練習

その映画はおもしろいです。　→　
その映画はおもしろい。

その辞書はよくないです。　→　
その辞書はよくない。

昨日のパーティーは楽しかったです。　→　
昨日のパーティーは楽しかった。

昨日は寒くなかったです。　→　
昨日は寒くなかった。

二类形容词谓语形式简体

敬体 简体
簡単です 簡単だ
簡単ではありません 簡単 では/じゃあ ない
簡単でした 簡単だった
簡単ではありませんでした 簡単 では/じゃあ なかった

名词谓语形式简体 [和二类形容词相同]

敬体 简体
雨です 雨だ
雨ではありません 雨 では/じゃあ ない
雨でした 雨だった
雨ではありませんでした 雨 では/じゃあ なかった

練習

カリナさんは絵が上手です。　→　
カリナさんは絵が上手だ。

チョコレートが好きではありません。　→　
チョコレートが好きじゃない。

きのうは晴れでした。　→　
きのうは晴れだった。

先週の土曜日は暇じゃありませんでした。　→　
先週の土曜日は暇じゃなかった。

动词谓语形式简体

敬体 简体
动词连用形 + ます 动词基本形
动词连用形 + ません 动词ない形
动词连用形 + ました 动词た形
动词连用形 + ませんでした 动词なかった形

練習

朝７時に起きます。　→　
朝７時に起きる。

タバコを吸いません。　→　
タバコを吸わない。

のどが渇きました。　→　
のどが渇いた。

昨日テレビを見ませんでした。　→　
昨日テレビを見なかった。

简体的会话

  • 1）简体疑问句中，一般省略句末的「か」，读升调。
    コーヒーを飲む?(↗)　→　コーヒーを飲みますか。
    うん、飲む。(↘)　→　はい、飲みます。

    • example
      あしたうちにいる?
      ううん、いない。

    • 練習
      パスポートを持っていますか(ううん)　→
      パスポートを持っている?
      ううん、持っていない。

      けさ新聞を読みましたか(うん)　→
      けさ新聞を読んだ?
      うん、読んだ。

      東京は雨でしたか(ううん)　→
      東京は雨だった?
      ううん、雨じゃなかった。

      富士山はどうでしたか(きれい)　→
      ふじさんはどうだった?
      きれいだった。

      映画はおもしろかったですか(ううん、全然)　→
      映画はおもしろかった?
      ううん、全然おもしろくなかった。

  • 2）名词和二类形容词的疑问句中，省略「だ」，或加上终助词缓和语气。
    京都は静か?(↗)　→　男女通用
    うん、静か/静かだ/静かだよ　→　男性用语
    うん、静か/静かよ　→　女性用语
    ううん、静かじゃないよ　→　男女通用

  • 3）简体句中若从前后关系可判断句意时，多省略助词。
    パン(を)食べる?
    そこに机(が)ある?
    この携帯(は)高いね。
    これから図書館[へ]行かない?
    【不能省】 で、に、と、から、まで

    • example
      ゴルフ(を)したこと(が)ある。
    • 練習
      東京へ行ったことがあります　→
      東京(へ)行ったこと(が)ある
      ピアノが上手じゃありません　→
      ピアノ(が)上手じゃない

  • 4）简体句中，「动词している」的「い」常常省略。
    今勉強して(い)る?
    うん、勉強して(い)る。

    • example
      今雨が降って(い)る?
      うん、降ってる。
    • 練習
      佐藤さんの住所を知っています（ううん）　→
      佐藤さんの住所を知って(い)る?
      ううん、知らない。

練習

A：冰激凌喜欢吗?
A：アイスクリーム、好き?
B：不，不太喜欢。你呢?
B：ううん、あんまり好きじゃないわ。あなたは?
A：非常喜欢哟。
A：大好き(だ)よ。
B：不吃太多比较好哦。会胖哟。
B：たくさん食べないほうがいいよ。太るよ。
