新标日初上09语法

投稿日 2022-01-06 編集日 2022-04-05 カテゴリ 新标日 単語数： 1.2k 読書の時間 ≈ 2 分

形容词

一类形容词(形1)

  • 形容词・イ形容词
    熱い　寒い　

  • 用法

    • 作谓语
      • 富士山は高いです。
    • 作修饰名词
      • 富士山は高い山です

二类形容词(形2)

  • 形容动词・ナ形容词
    綺麗[だ]　有名[だ]　

一类形容词 + 名[定语]

中国は　広い　国です。

  • 一类形容词[～い] + 名词
    • おいしい料理　熱いコーヒー　楽しい夏休み　
    • 広い教室　古い車　高いカバン　

example

毎日お茶を飲みます。(熱い)　→　
毎日熱いお茶を飲みます。

練習

昨日カバンを買いました。(安い)　→　
昨日安いカバンを買いました。

先週ケーキを買いました。(おいしい)　→　
先週おいしいケーキを買いました。

一类形容词作谓语

  • 1.非过去时肯定形式

    • 一类形容词「い」 + です
      • 四川料理は　辛いです。
    • example
      • あの女の子はかわいいです。
      • 新しい先生はおもしろいです。

  • 2.非过去时否定形式

    • 一类形容词「い」 + くないです/くありません
      • 今日は　寒く　ないです。
    • example
      • 学校は駅から近くないです。
      • この薬は苦くありません。

  • 3.过去时肯定形式

    • 一类形容词「い」 + かったです
      • 先週は　忙しかったです。
    • example
      • 昨日は暑かったです。
      • 旅行は楽しかったです。

  • 4.过去时否定形式

    • 一类形容词「い」 + くなかったです/くありませんでした
      • 昨日は　暑く　なかったです。
    • example
      • この時計は高くなかったです。
      • 昨日の試験は易しくなかったです。

  • 5.特例「いい」

    • 非过去时肯定形式　→　いいです
    • 非过去时否定形式　→　よくないです/よくありません
    • 过去时肯定形式　→　よかったです
    • 过去时否定形式　→　よくなかったです/よくありませんでした
    • example
      • 昨日の天気はよかったです。

example

この靴は新しいですか。(はい)
はい、新しいです。

昨日は暑かったですか。(いいえ)
いいえ、暑くなかったです。

練習

この本はおもしろいですか。(はい)　→　
はい、おもしろいです。

部屋は狭いですか。(いいえ)　→　
いいえ、狭くないです。

旅行は楽しかったですか。(はい)　→　
はい、楽しかったです。

昨日は寒かったですか。(いいえ)　→　
いいえ、寒くなかったです。

一类形容词时态变化表

非过去(现在・将来) 过去
普通 肯定 熱いです 熱かったです
否定 熱くないです(熱くありません) 熱くなかったです(熱くありませんでした)
特例 肯定 いいです よかったです
否定 よくないです(よくありません) よくなかったです(よくありませんでした)

表示程度的副词

この料理はとてもおもしろいです。

中文 日文
很，非常 とても・たいへん
一点儿 少し・ちょっと
不太~ あまり～ません
根本不~ 全然～ません

example

日本語はとてもおもしろいです。
四川料理はちょっと辛いです。
昨日はあまり忙しくなかったです。
新しい家は全然広くないです。

練習

A：星期日买了什么?
A：日曜日、何を買いましたか。
B：买了新鞋子和有意思的书。
B：新しい靴とおもしろいほんを買いました。
A：看电影了吗?
A：映画を見ましたか。
B：看了。一点儿也没意思。特别无聊。
B：見ました。全然おもしろくなかったです。とてもつまらなかったです。
