新标日初上16语法

投稿日 2022-01-12 編集日 2022-03-23 カテゴリ 新标日 単語数： 1.3k 読書の時間 ≈ 2 分

16 ホテルの　部屋は　広くて　明るいです

形１て[并列]

一类形容词：形1「～い」 → 「くて」
おいしい → おいしくて
かわいい → かわいくて いい → よくて

あの先生は優しくてきれいです。
そこは広くて明るくて静かな図書館です。

example

李さんはどんな人ですか。(若い/元気) →
若くて元気な人です。

練習

李さんはどんな人ですか。(髪が短い/かわいい) →
髪が短くてかわいい人です。

李さんはどんな人ですか。(頭がいい/まじめ) →
頭がよくてまじめな人です。

形2て[并列]

二类形容词：形2 +「で」
静か → 静かで
きれい → きれいで

李さんはまじめで元気です。
田中さんは親切できれいで優しい先生です。

example

どんな所ですか。(静か/きれい) →
静かできれいな所です。

練習

どんな所ですか。(便利/にぎやか) →
便利でにぎやかな所です。

どんな町ですか。(きれい/静か) →
きれいで静かな町です。

example

旅行/天気がいい/楽しい →
旅行はどうでしたか。
天気がよくて、楽しかったです。

練習

パーティー/にぎやか/おもしろい →
パーティーはどうでしたか。
にぎやかで、おもしろかったです。

名词で[并列]

名词：名词＋「で」
日曜日 → 日曜日で

  • 相同主题
    田中さんは日本人で、北京大学の留学生です。
  • 不同主题
    田中さんは留学生で、森さんは社員です。

example

スミスさん/旅行会社の社員/営業部の部長 →
スミスさんは旅行会社の社員で、営業部の部長です。

練習

李さん/中国人/JC企画の社員 →
李さんは中国人で、JC企画の社員です。

キムさん/韓国人/研修生 →
キムさんは韓国人で、研修生です。

动词ています

森さんは車を持っています。

回顾
李さんは今教室でべんきょうしています。[正在进行]
状态性动词ています → 表示结果状态的存续

メガネをかけます　メガネをかけました　メガネをかけています
車を買います　車を買いました　車を持っています

  • 状态性动词
    持ちます 結婚します 住みます 知ります

    • 持ちます
      李さんは車を持っていますか。
      はい、持っています。
      いえ、持っていません。

      荷物を持ってください。

    • 結婚します
      もう結婚していますか。
      はい、もう結婚しています。
      いいえ、まだ結婚していません。

      あの女はきれいですね。
      だめですよ。あの人はもう結婚しましたよ。

    • 住みます
      あなたはどこに住んでいますか。
      北京に住んでいます。

      今、東京に住んでいますか。
      はい、東京に住んでいます。
      いいえ、東京に住んでいません。

    • 知ります
      森さんを知っていますか。
      はい、知っています。
      いいえ、知りません。

[持ちます] 練習


何を持っていますか。
家を持っています。


何を持っていますか。
何も持っていません。

練習

A：已经结婚了吗?
A：もう結婚していますか。
B：没有，还没有结婚。和父母一起住在北京。
B：いいえ、まだ結婚していません。両親といっしょに北京に住んでいます。
A：有车吗?
A：車を持っていますか。
B：有一辆设计很别致白色的车。
B：デザインがよくて白い車を持っています。
A：认识JC企画的小李吗?
A：JC企画の小野さんを知っていますか。
B：不，不认识。是个什么样的人?
B：いいえ、知りませんよ。どんな人ですか。
A：是个很漂亮、可爱、又温柔的人。
A：きれいでかわいくて優しい人ですよ。
B：是吗。是个不错的人啊。
B：そうですか。いい人ですね。
