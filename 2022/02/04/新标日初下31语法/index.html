新标日初下31语法

投稿日 2022-02-04 編集日 2022-02-17 カテゴリ 新标日 単語数： 1.6k 読書の時間 ≈ 3 分

31 このボタンを押すと、電源が入ります

动词 + と

接续：动词基本形・ない形 + と
このボタンを押すと、電源が入ります。
と： 接续助词，【假定条件】，表示事情实现的条件、前提。

  • (1)表示客观真理、恒常状态。前后是必然关系。译为"一...就"。

    春になると、暖かくなります。
    練習しないと、上手になりません。

  • (2)表示特定任务的主观习惯。译为"一...就(总是)..."。

    ご飯を食べると、すぐ眠くなります。
    家に帰ると、すぐお風呂に入ります。

  • (3)表示由于某种行为而发现了新的状况，后半句多为过去时态。译为"一...就(发现)..."。

    カーテンを開けると、美しい山を見えました。
    ドアを開けると、大きいな犬がいました。

  • (4)表示地理位置。多用于指路。译为"一...就(有)..."。

    その橋を渡って右へ曲がると、銀行があります。
    この道をまっすぐ行くと、コンビニがあります。

补充说明

后半句不能是自己的意志、愿望、命令、劝诱等内容。

【意志】夏になると、海へ行こう。
【愿望】夜になると、帰りたいです。
【命令】夜になると、早く帰ってください。
【劝诱】夜になると、食事に行きましょう。

example＆練習

夜になります/急に気温が下がります →
夜になると、急に気温が下がります。

薬を飲みません/病気が治りません →
薬を飲まないと、病気が治りません。

難しい本を読みます/寝ることができなくなります →
難しい本を読むと、寝ることができなくなります。

信号のある交差点を右に曲がります/左にスーパー →
信号のある交差点を右に曲がると、左にスーパーがあります。

动词基本形・ない形 + ことがあります

表示有时会或不会有某种行为或情况发生。译为”有时(不)会…”。

  • 动词基本形 + ことがあります：译为"有时会..."。
  • 动词ない形 + ことがあります：译为"有时不会..."。
呼应使用的副词 + たまに・時々

example＆練習

このパソコンは、/たまにフリーズしまし →
このパソコンは、たまにフリーズすることがあります。

父はたまに怒ります →
父はたまに怒ることがあります。

森さんは時々寝ません →
森さんは時々寝ないことがあります。

补充说明

  • 动词た形ことがあります 曾经…
    家族とアメリカへ旅行に行ったことがあります。
  • 动词た形ことがありません 从未…
    日本りょりを食べたことがありません。
  • 动词基本形ことがあります 有时会…
    時々寝坊することがあります。
  • 动词ない形ことがあります 有时不会…
    たまに朝ご飯を食べないことがあります。

形容词的副词性用法

早く家に帰りましょう。

  • 接续1：形1「い」+ く + 动词
    難しい → 難しく　うまい → うまく　いい → よく
  • 接续1：形2 + に + 动词
    簡単 → 簡単に　上手 → 上手に

example

子供たちはおいしくご飯を食べています。
李さんは東京でとても楽しく過ごしました。
李さんは先生に丁寧にお礼を言いました。
張さんは日本語を上手に話しています。

简体句 + でしょうか(↘)

  • 接续1：动词/形1简体 + でしょうか
  • 接续1：名词/形2简体「だ」 + でしょうか 表示较委婉地提问。礼貌程度较高。

李さんは来ますか。 → 李さんは来るでしょうか。
これは日本のお菓子ですか。 → これは日本のお菓子でしょうか。

example＆練習

この計画に、部長は賛成してくれるでしょうか。
先週の試験は難しかったでしょうか。
東京は交通が便利でしょうか。
あそこに電話をかける人は王さんの先生でしょうか。

馬さんはもう帰りましたか →
馬さんはもう帰ったでしょうか。

この料理は辛いですか →
この料理は辛いでしょうか。

李さんはスイカが好きですか →
李さんはスイカが好きでしょうか。

これは小野さんの傘ですか →
これは小野さんの傘でしょうか。

練習

1.不快点出门的话，就会赶不上电车了。
早く出かけないと、電車を間に合いませんよ。
2.平时都是坐电车去学校的，但有时也会坐巴士去。
いつも電車で学校へ行きますが、たまにバスで行くことがあります。
3.睡觉之前，有时也会不泡澡。
寝る前に、たまにお風呂に入らないことがあります。
4.小野没有来吗？
小野さんは来なかったでしょうか。
