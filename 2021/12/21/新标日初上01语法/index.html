新标日初上01语法

投稿日 2021-12-21 編集日 2022-04-05 カテゴリ 新标日 単語数： 875 読書の時間 ≈ 1 分

01 李さんは中国人です

わたし　は　中国人です

  • は：提示助词，读作「わ」，提示主题
  • です：判断助动词 “是”
  • … は …　です；… 是 …

example

わたし/学生　 → 　わたしは学生です。
わたし/会社員　 → 　わたしは会社員です。
李さん/留学生　 → 　李さんは留学生です。
わたし/王　 → 　わたしは王です。

李さんは中国人です。
李さんは会社員です。

ヂュポンさんはフランス人です。
ヂュポンさんは教授です。

わたし　は　中国人　では　ありません

  • 口语：ではありません　 → 　じゃありません

example

わたし/日本人　 → 　わたしは日本人ではありません。
森さん/課長　 → 　森さんは課長ではありません。
わたし/王　 → 　私は森では(じゃ)ありません。

李さん　は　中国人　ですか

  • か：疑问终助词
  • 答：はい、「李さんは」中国人です。
  • 答：はい、そうです。
  • 答：いいえ、「李さんは」日本人ではありません。中国人です。
  • 答：いいえ、違います。中国人です。

example

Q:あなたは学生ですか。
A:はい、「わたしは」学生です。
A:はい、そうです。
A:いいえ、「私は」学生ではありません。会社員です。
A:いいえ、違います。店員です。

Q:「あなたは」李さんですか。
A:はい、「私は」李です。
A:はい、そうです。
A:いいえ、「私は」李ではありません。緑です。
A:いいえ、違います。王です。

Listen

李さんは中国人ですか。
はい、そうです。
李さんは学生ですか。
いいえ、違います。

小野さんは日本人ですか。
はい、そうです。
小野さんは学生ですか。
いいえ、違います。

金さんはちゅうごくじんですか。
いいえ、違います。

スミスさんはアメリカ人ですか。
はい、そうです。

李さん　は　 JC 企画　の　社員です

  • の：格助词。表示从属的机构、国家、属性。”的”
    • JC 企画の社員[机构]
    • 日本で留学生[国家]
    • 大学の先生[属性]
  • 补充：日语中，不论名词之间是什么关系，一般加 「の」
    • 我父亲
      • わたし父　　 ×
      • わたしの父　 √

練習

Q:田中先生是公司职员吗?
田中さん は 会社員 ですか。
A:不，不是公司职员。是大学教授。
いいえ、違い ます。大学 の 教授 です。
Q:是东京大学的教授吗?
東京大学 の 教授 ですか。
A:是，是东京大学的教授。
はい、そう です。
